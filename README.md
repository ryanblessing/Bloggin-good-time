# Bloggin-good-time
Bloggin good time is a website that allows you to: 

- create a your own account

- post blogs

- create, update or delete post as time goes on


## Installation
For this app you need to install:

- mysql2

- sequelize

- express

- dotenv

- bcrypt

- connect-session-sequelize

- express-session

- handlebars
